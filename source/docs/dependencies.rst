.. _dependencies:

Dependencies
============

This project requires the following dependencies:

- `tk`
- `Pillow`
- `numpy`
- `requests`
- `matplotlib`
- `Sphinx`
- `sphinx-rtd-theme`
- `docutils`
- `Pygments`
- `Jinja2`

.. note::
    To install the dependencies, please refer to the :ref:`usage` section..
