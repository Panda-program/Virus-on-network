Setup & Usage
=============

Setup
-----

To install the Virus on Network project, follow these steps:

1. Clone the repository:

    .. code-block:: bash

        git clone https://github.com/Panda-program/Virus-on-network

2. Install dependencies:

    .. code-block:: bash

        pip install -r requirements.txt

.. note::
    For more information on dependencies, please refer to the :ref:`dependencies` section.


Usage
-----
To run the project, execute the following command:

.. code-block:: bash

   python main.py
